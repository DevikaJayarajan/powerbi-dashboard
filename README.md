# 📊 Global YouTube Analytics Dashboard (Power BI)

## Overview

This project presents an interactive **Power BI dashboard analyzing global YouTube channel performance**. The dashboard provides insights into views, subscriber growth, content categories, and estimated earnings across major YouTube channels.

The goal of this project is to demonstrate **data analysis, visualization, and storytelling using Power BI** to extract meaningful insights from large-scale social media data.

---

## Key Metrics

The dashboard highlights several key performance indicators:

* **Total Views:** 11T
* **Views in Last 30 Days:** 152 Billion
* **Average Views per Upload:** 1.17M
* **Earliest Channel Creation Date:** June 22, 2005

These metrics provide a quick overview of global YouTube platform activity.

## Insights from the Analysis

Some key observations from the dashboard include:

* Music and entertainment channels dominate global viewership.
* Large channels like **T-Series and MrBeast maintain extremely high subscriber growth.**
* Categories like **Gaming and Entertainment generate strong engagement per upload.**
* Subscriber growth does not always correlate directly with views per upload.
